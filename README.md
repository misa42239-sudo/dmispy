**Developed by Wolf (Dmimaks) Discord volkchonek

Powerful tool for digital reconnaissance and target tracking.

## 🚀 Features
* **[1] Social Scan:** Find profiles by nickname across 10+ platforms.
* **[2] TG Probe:** Deep analysis of Telegram accounts (ID, Status, Bio).
* **[3] GitHub Recon:** Instant access to developer profiles.

## 🛠 Installation
1. Clone this repository.
2. Install dependencies: `pip install requests colorama telethon`
3. Add your Telegram API credentials (API_ID & API_HASH) inside `dmispy.py`.
4. Run the script: `python dmispy.py`
